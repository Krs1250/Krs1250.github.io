# 线性表
*特征：*  
*- 有且只有一个"第一元素"*  
*- 有且只有一个"最后元素"*  
*- 除第一元素之外，其他元素都有唯一的直接前趋*  
*- 除最后元素之外，其他元素都有唯一的直接后继*

## [链表_LinkList(LinkedList)](./Linear_List/Linked_List/Linked_List.md)
*特征：线性表的链式存储方式，是指用一组任意的存储单元来存储线性表的结点。*  
*优点：能够方便地增加和删除线性表中的元素*  
*缺点：使随机存储、获取直接前趋和直接后继变得较为复杂*
